# Rhino FCP Examples

This folder contains examples for using different capabilities of the Rhino Federated Computing Platform, organized as a progressive learning path from data ingestion through to advanced platform features.

## Learning Path
```
START HERE
    │
    ▼
01_data-ingestion        "Import data into FCP"
    │                    SFTP · Mounted Storage · S3/GCP
    ▼
02_data-preprocessing    "Clean and transform"
    │                    DICOM · CSV merging · splits
    ▼
03_federated-learning    "Run federated training of ML models across Rhino FCP sites"
    │                    Hello World → Regression → Imaging → XGBoost
    ▼
04_fine-tuning           "Specialize large models"
    │                    BioNeMo LLM · MONAI · Flower
    ▼
05_sdk-and-analytics     "Query, analyze and visualize results"
    │                    EDA · Metrics · FHIR · SQL · Streamlit
    ▼
06_platform-features     "Advanced platform capabilities"
                         GPU · Encrypted code · Interactive containers
```
